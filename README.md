# DormFit

DormFit 是一个基于 AI Coding 快速开发的宿舍健身 Web 应用，面向宿舍空间有限、器械不足但希望持续训练的学生用户，提供训练计划、动作指导、每日打卡与训练数据反馈等功能。

## 项目亮点

- 基于 Codex 通过自然语言完成需求拆解、页面生成与多轮迭代
- 围绕“计划 - 执行 - 反馈”设计完整训练闭环
- 支持训练计划查看、动作库、每日打卡、累计训练量与近 7 日趋势展示
- 使用浏览器本地存储保存训练记录，无需后端即可完成原型验证

## 核心功能

- 训练计划：提供拉、推、腿、休息 4 天循环训练安排
- 动作库：展示动作名称、训练提示与示意图
- 每日打卡：记录动作组数与次数
- 数据看板：展示累计训练量、连续训练天数、总组数与近 7 日趋势

## 技术栈

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Chart.js
- localStorage

## AI Coding 说明

本项目使用 Codex 参与从 0 到 1 的原型开发过程，包括：

- 将初始想法拆解为页面与功能模块
- 生成页面结构、交互逻辑与样式代码
- 根据使用场景进行多轮调整与优化
- 辅助完成从产品构想到可交互原型的快速落地
